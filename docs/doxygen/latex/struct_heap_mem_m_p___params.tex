Size of shared buffer

This is the size of the buffer to be used in the HeapMemMP instance. The actual buffer size in the created instance might actually be less than the value supplied in 'sharedBufSize' because of alignment constraints.

It is important to note that the total amount of shared memory required for a HeapMemMP instance will be greater than the size supplied here. Additional space will be consumed by shared instance attributes and alignment-\/related padding. \index{HeapMemMP\_\-Params@{HeapMemMP\_\-Params}!gate@{gate}}
