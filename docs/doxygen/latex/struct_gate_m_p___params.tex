Name of this instance.

The name (if not NULL) must be unique among all GateMP instances in the entire system. The name does not have to be in persistent memory. The supplied string is copied into persistent memory. If no name is supplied, the instance cannot be opened calling \hyperlink{_gate_m_p_8h_acefd091d723ca21a0a02e1e1d5600fc8}{GateMP\_\-open}. The max length of the name is defined by the maxNameLen of the NameServer instance. \index{GateMP\_\-Params@{GateMP\_\-Params}!regionId@{regionId}}
